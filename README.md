# Домашнее задание 1.4 — Docker Compose

Задание *необязательное*: docker-compose для Django-проекта с БД (backend, postgres, nginx).

## Структура

- **backend** — Django + Gunicorn
- **postgres** — PostgreSQL
- **nginx** — reverse proxy и статика

Запуск: `docker-compose up -d`. Приложение доступно на порту 80.
